# Python Card Game - PyCards

## Deck Composition (54 cards)

1. **Primitive Types** (11 cards)
   - Int (x3) - 1 point
   - Float (X3) - 1 point
   - String (x3) - 2 points
   - Bool (x2) - 1 point


2. **Data Structures** (12 cards)
   - List (x3) - 2 points
   - Tuple (x3) - 3 points
   - Dictionary (x3) - 4 points
   - Set (x3) - 4 points


3. **Keywords** (27 cards)
   - if (x3) - 3 points
   - elif (x2) - 3 Points
   - else (x2) - 3 Points
   - try (x2) - 4 Points
   - except (x2) - 4 Points
   - finally (x2) - 4 Points
   - for (x2) - 5 points
   - while (x2) - 5 points
   - def (x2) - 6 points
   - return (x2) - 4 points
   - yield (x2) - 5 points
   - with (x2) - 5 points
   - break (x1) - 4 points
   - raise (x1) - 4 points


5. **Special/Trump Cards** (4 cards)
   - Comprehension (x2) - 6 points
   - Placeholder/Joker (x2) - Flexible
